# test_printf

To use this test, clone it in your repo, compile with your libftprintf.a

exemple : `gcc -Wall -Wextra test.c -g3 libftprintf.a && ./a.out`

by default, you will run classics tests
options works with av1 
    ex : `./a.out a`
    - a for all tests
    - d for %d
    - s for %s
    - c for %c
    - p for %p
    - x for %x
    - X for %X
    - u for %u
    - o for %%
    - m fot mixed tests
